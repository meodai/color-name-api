Color Name API

Rest API that returns a bunch of color names for a given color-value. It comes with different open-source name lists. This page is meant to be a playground for the API. You can test the API and see how it works.

API Request URL Builder

Interact with the elements below to build your API request URL.

$ curl https://api.color.pizza/v1/ ?values= &list= &noduplicates=
