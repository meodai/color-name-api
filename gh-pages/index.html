Color Name API

The Color Name API is a powerful tool that provides a variety of color names based on a given color value. It leverages multiple open-source name lists to deliver accurate and diverse results. This page serves as an interactive playground where you can explore and test the API in action.

$ curl https://api.color.pizza/v1/ ?values= &list= &noduplicates=

About

      

        This interactive tool lets you explore the API's capabilities in a hands-on way. 
        While not a comprehensive documentation, you can find more details about the API on its 
        GitHub page or in the 
        official API reference.
      

      

        The falling particles are powered by Matter.js. 
        Each particle represents a color name fetched from the API in real-time using Socket.IO.
      

      

        As you can imagine from the number of requests, running this service is not free. 
        If you find this tool useful, consider supporting the project by donating via Buy Me a Coffee.
